# Vorzai — 电商桌面 AI 助手

Vorzai 是一款面向电商从业者的桌面级 AI Agent，集成多渠道电商数据、智能运营决策、营销自动化与业务洞察于一体，帮助商家在日常运营中高效决策。

## 核心功能

- 多平台电商数据聚合与实时监控
- AI 驱动的商品选品与定价建议
- 智能营销活动规划与执行
- 订单处理与库存管理自动化
- 销售数据可视化与经营分析

## 下载安装

请从 [GitHub Releases](https://github.com/kule-2025/vorzai-public/releases) 下载最新安装包。

## 技术栈

- **桌面框架**：Electron + React + TypeScript
- **AI 引擎**：多模型适配（LLM Adapter 架构）
- **数据存储**：本地 SQLite（离线可用）
- **更新机制**：无感自动更新（GitHub/Gitee 双源）

## 构建

```bash
git clone <private-repo-url>
npm install
npm run build
```
